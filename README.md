
# GEE Chlorophyll Monitor Backend

## Overview
This project provides a FastAPI backend integrated with Google Earth Engine for chlorophyll monitoring.

### Features
- Generate chlorophyll index tiles
- Compute composites from Sentinel-2
- API endpoints for visualization and analysis
